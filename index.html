обо мне

Никитин Павел (г. Красноярск) - html верстальщик, вебразработчик.

Моя специализация: Создание сайтов и Landing Page (фиксированная, резиновая, адаптивная), без посадки на CMS. Верстка с использованием фреймворка bootstrap.

В процессе разработки пишу легкочитаемый, прокомментированный в нужных местах код, который будет понятен другому разработчику. В работе использую Webpack, препроцессоры sass и less, javascript библиотеки jquery и reactjs.
